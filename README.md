# 🏠 租房对比助手

一个帮助你系统记录、评估和横向对比候选房源的单页工具。所有数据保存在浏览器本地（LocalStorage），无需后端、无需登录，打开即用。

## ✨ 功能特性

### 📋 房源记录
为每套房源记录 30+ 个维度的信息，覆盖租房决策的方方面面：

| 分类 | 记录项 |
|------|--------|
| 💰 费用 | 月租金、押金（月数）、中介费、物业费 |
| 🏠 户型 | 面积、户型、楼层/总层数、朝向、装修成色 |
| 🚇 交通 | 距地铁步行分钟、地铁线路、通勤时间 |
| 🔌 电器 | 空调、洗衣机、冰箱、热水器、燃气灶等 12 件 |
| ☀️ 环境 | 采光评分、噪音等级 |
| 🛍️ 配套 | 距超市距离、餐饮丰富度、医疗资源 |
| 🌿 小区 | 绿化评分、安保门禁、停车位、楼龄 |
| 🤝 房东 | 房东印象、最短租期、是否允许养宠/做饭 |
| ⭐ 综合 | 综合满意度评分、自由备注 |

### ⚖️ 横向对比
- 在房源列表勾选想对比的房源（最多 4 套）
- 一张表格展示所有维度的并排对比
- **自动高亮最优项**（租金最低、面积最大、地铁最近等）
- 顶部显示加权综合评分排行榜

### 🏆 综合评分
根据各维度星级评分自动计算加权综合分（满分 5 分）：
- 综合满意度（权重 ×3）
- 交通便利度（×2）
- 周边配套（×1.5）
- 家电状况（×1.5）
- 采光（×1）、绿化（×1）、房东印象（×1）

### 📤 导出 / 导入
- 一键将全部房源数据导出为 `.json` 文件（方便备份）
- 支持导入已有 JSON 文件（可选择合并或覆盖）

### 🔀 列表排序
支持按以下方式排序：
- 添加时间（默认）
- 综合评分从高到低
- 租金从低到高
- 地铁距离从近到远

## 🚀 使用方法

直接用浏览器打开 `index.html` 即可，无需任何安装或配置。

```bash
# 如需本地服务器访问（可选）
python3 -m http.server 8765
# 然后访问 http://localhost:8765
```

---

## 💾 数据存储

所有数据保存在浏览器的 **LocalStorage** 中，key 为 `rental_houses_v2`。

- ✅ 数据仅存在本机浏览器，不会上传
- ⚠️ 清除浏览器缓存会导致数据丢失，建议定期使用「导出」功能备份
